A compact RF-focused device built for long-range communication and signal experimentation.

## Hardware

- ESP32-C3 Supermini
- Dual NRF24L01 + PA/LNA modules
- 2000mAh battery (TPS63020 + TP4056 + INA226)
- 0.96” OLED display
- IPEX → SMA external antenna
- Resin enclosure with transparent lid

## Features

- NRF communication tools + chat
- RF experimentation tools
- Signal jamming (research/educational use)
- Portable, high-efficiency power design
