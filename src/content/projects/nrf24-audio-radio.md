A real-time wireless audio communication system, built as a matched pair.

## Hardware (per unit)

- ESP32-C3 Supermini
- NRF24L01 + PA/LNA
- INMP441 I2S microphone
- I2S 3W amplifier
- 0.96” OLED + buttons

## Features

- Half-duplex walkie-talkie audio
- ~800m line-of-sight range
- Preset messaging system
- Paging / alert mode
