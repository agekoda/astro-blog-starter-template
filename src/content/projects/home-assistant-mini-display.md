A compact smart home display with a clean UI.

## Hardware

- ESP32-C3 Supermini
- 2.8” 320×240 TFT
- Capacitive touch buttons
- VEML7700 ambient light sensor

## Features

- MQTT integration with Home Assistant
- Automatic brightness adjustment
- Customisable UI + settings menu
